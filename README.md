# Todoie
## Another Todo list app :)
### Features:
1. Authentication using [PassportJs](https://github.com/jaredhanson/passport) and [Bcrypt](https://github.com/kelektiv/node.bcrypt.js#readme)
2. Todolist
3. Grouping of todolists

### Tech used:
Frontend:
1. React & Redux
2. Bootstrap
3. React Router

Backend:
1. NodeJs & Express
2. PassportJs
3. Bcrypt
4. Cookie-session
5. Mongoose (MongoDB)
